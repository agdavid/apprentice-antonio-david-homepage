= AntonioDavidSaysHey!

== About

This homepage was created in response to the rad StandardCo.de challenge for their apprenticeship.

There were a few general requirements, such as: (1) configure custom routes; (2) use links; (3) create a simple databse for questions and answers.  For specific details please look at the file in this repo StandardCodeSPECS.md.

== Usage
The application can be run locally on your localhost:
1. Clone it (<tt>git clone git@github.com:agdavid/apprentice-antonio-david-homepage.git</tt>)
2. Run bundler(<tt>bundle install</tt>)
3. Migrate the database (<tt>rake db:migrate</tt>)
4. Seed the database (<tt>rake db:seed</tt>)
5. Run your server! (<tt>rails s</tt>)
